# video-rpc

> Discord RPC / Rich Presence example for YouTube

> You can view the video [here](https://www.youtube.com/watch?v=Hl-MSPlR8WU)

## Installation Instructions

```bash
# install dependencies
$ npm install

# start client
$ node main.js
```
